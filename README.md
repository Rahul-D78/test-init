## Project setup from scratch :

* Initiate your project with vitejs :
```$ npm init @vitejs/app```

* Select the vanila javascript option : 

* Install `Jest` :
```$ npm i jest --save-dev```

* First failing test :
```$ npm run test```

* For type safety :
```
npm i @types/jest --save-dev

create a jsconfig.json file in your root directory

{
  "typeAcquistion": {
      "include": [
         "jest"
      ]
   }
}
```
